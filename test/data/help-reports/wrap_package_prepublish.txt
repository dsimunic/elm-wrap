Usage: wrap package prepublish SOURCE_PATH [OPTIONS]

Determine which files would be published from a package.

Arguments:
  SOURCE_PATH           Path to the package directory

Options:
  -h, --help            Show this help message
  --git-exclude-extras  Generate .gitattributes to exclude extras from git archive
  --generate-docs-json  Generate docs.json file in package root
  -f, --overwrite       Overwrite existing docs.json (use with --generate-docs-json)
